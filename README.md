## ✨ 功能

| 工具 | 说明 |
|------|------|
| 🔍 **布尔搜索生成器** | 可视化拼接 AND/OR/NOT 语句，精准搜索候选人 |
| 🔑 **关键词拓词器** | 输入行业/角色，生成高权重搜索关键词 |
| 📝 **资料优化器** | 优化 LinkedIn 个人资料，提升招聘顾问品牌形象 |
| 📄 **JD 分析器** | 分析职位描述，提取技能关键词，简历匹配度检测 |
| 💼 **职位探索器** | 探索市场职位行情，了解技能要求分布 |

## 🎯 适用人群

- **招聘顾问 / 猎头**：精准搜索候选人，提升人脉质量
- **HR / 人才寻访**：高效筛选，降低沟通成本
- **RPO 服务商**：批量搜索，提升交付效率
